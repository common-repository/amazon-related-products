Add contextual Amazon products into your blog based on the tags or defined keywords. 
== Description ==
Monetize your blog adding Amazon contextual products to your posts and pages. Using this plugin you can add products images, titles and prices automatically or using shortcodes and widgets based on the tags attached to the content or using defined keywords.

Special Features:

*   Easy to use: Very simple and straightforward option page.
*   The Amazon products are displayed in real time: The products never expired.
*   Allows multiple ads per page.
*   The products can be added automatically or manually using shortcodes or widgets.
*   Multiple unit types like list, grids.

== Installation ==

1. Upload 'Amazon Related Products' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Amazon Related Products Options page under settings and save your preferred options.
4. Then you are able to insert the related products automatically or using the shortcode.

== Frequently Asked Questions ==

= Do I need to change the products links? =

No, once you add the shortcode into your post or widget the product will come directilly from amazon based on your tags or defined keywords. So you will show fresh and new content based on Amazon inventory. 

= How can I get my Amazon Associates ID and secret keys? =

In order to use the plugin and get revenue from your content you need to register on [Amazon Associates](https://affiliate-program.amazon.com/) and sign up in [AWS Products - Amazon Web Services](https://aws.amazon.com/products/).
